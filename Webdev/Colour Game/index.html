How to play:

The point of this game is to match colours to a selected colour. Start by pressing the button.
To select a colour you must first select a tile. The colour of the selected tile will show above the squares.
You must select the squares that are the same colour to this. You can select a different scoloured square to match a different colour.
Once you have matched all squares to the correct colour - YOU WIN.
You can restart the game again by pressing the button.
